<summary><b>🔐 Quant — Secure Communication Protocol (Rust)</b></summary>
<br>

Enterprise-grade secure communication protocol built from scratch in Rust. Designed with security-first principles at the protocol layer, not bolted on.

- Zero Trust architecture with mutual authentication
- Forward secrecy via ephemeral key exchange
- Role-Based Access Control (RBAC)
- Tamper-evident audit logging
- Secure client-server communication model

> Rust was chosen specifically for memory safety guarantees and its suitability for systems-level protocol engineering.
